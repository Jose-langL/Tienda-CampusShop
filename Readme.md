# 🛒  Tienda CampusShop

## 📖 Descripción
TIENDA-CAMPUSSHOP es una página web de e-commerce diseñada con un estilo minimalista y funcional. Su objetivo es ofrecer a los usuarios una experiencia sencilla y organizada para explorar productos, consultar ofertas y gestionar sus compras.

La aplicación permite:

Navegar por un catálogo con tarjetas de producto detalladas.

Identificar artículos en oferta o próximos a agotarse.

Revisar el historial de pedidos realizados.

Acceder a secciones clave como carrito, checkout y perfil de usuario.

Gracias a su estructura modular en HTML y CSS, el proyecto garantiza un diseño responsivo y una interfaz limpia que se adapta tanto a dispositivos móviles como de escritorio.

## 🚀 Características
- Catálogo responsivo con tarjetas de producto.
- Insignias de oferta y precios destacados.
- Botones para añadir al carrito.
- Diseño limpio y moderno con tipografía sans-serif.
- Paleta de colores corporativa y minimalista.

## 🛠️ Tecnologías utilizadas
- **HTML5** para la estructura.
- **CSS3** para estilos, animaciones y modales.
- **Responsive Design** para compatibilidad en dispositivos móviles y escritorio.
